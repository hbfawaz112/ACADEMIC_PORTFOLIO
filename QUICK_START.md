## 3Ô∏è‚É£ Enable GitHub Pages (2 minutes)

1. Go to **Settings** ‚Üí **Pages**
2. Under "Source":

3. After workflow completes, return and select `gh-pages`
4. Your site will be at: `https://YOUR-USERNAME.github.io/MFAWAZ_PORTFOLIO/`

## 4Ô∏è‚É£ Create Your First Post (3 minutes)

**Option A: On GitHub**
1. Go to `posts/` folder
2. Click "Add file" ‚Üí "Create new file"
3. Name it: `my-first-post.md`
4. Paste this template:

```markdown
---
title: My First Post
tags: [introduction]
date: 2025-11-13
summary: My first blog post!
---

## Hello World

This is my first post on my new portfolio.

I'll be writing about [your topics here].
```

5. Commit the file
6. Wait 1-2 minutes
7. Visit your blog page!

**Option B: Using Admin Panel**
1. Download repository (Code ‚Üí Download ZIP)
2. Extract and open `admin.html` in browser
3. Fill in the form with live preview
4. Click "Create Post on GitHub"
